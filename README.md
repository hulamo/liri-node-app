# liri-node-app

Este programa busca canciones o peliculas, se le debe de dar de entrada como primer argumento lo que quiere buscar [acción] y en segundo el texto a buscar [qbuscar]

node liri cancion love
node liri movie top gun

El programa no está 100% funcionamble porque para buscar películas hay que pagar la licencia, pero esta se puede sustituir.

Si el usuario quiere correr el programa tendrá que meter sus licencias en su propio keys.js
